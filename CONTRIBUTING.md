# Contributing

Thanks for contributing to `zpe-taste`.

## Scope

This repository carries a narrow evidenced negative. Contributions should improve clarity, reproducibility, packaging, testing, or documentation without inflating the scientific claim.

## Before Opening A Pull Request

1. Open an issue if the change alters public wording, proof anchors, or validation behavior.
2. Keep claims traceable to committed artifacts in `proofs/` and `validation/`.
3. Run the local verification steps from `README.md`.

## Contribution Rules

- Preserve the public claim boundary in `README.md` and `PUBLIC_AUDIT_LIMITS.md`.
- Do not add local machine paths, private references, or unpublished research notes.
- Keep changes atomic and evidence-backed.
