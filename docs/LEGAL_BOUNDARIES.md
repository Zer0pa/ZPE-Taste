## Public Scope

This repository publishes a bounded scientific result and the supporting public-safe artifacts required to verify it. It does not publish private process materials or unpublished research history.

## No Scope Inflation

Nothing in this repository should be read as a claim about all possible biological taste representations or as a claim that future taste work cannot succeed with a different native object.
